# Plu-Ts

Plu-ts streamlines Cardano DApp development by allowing developers to write smart contracts in TypeScript. It produces resource-efficient smart contracts through advanced compiler optimizations and strict type checking. By using a highly popular language like TypeScript for writing smart contracts it minimizes context-switching and makes it faster for devs to get started.
