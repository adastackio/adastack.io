# Off-Chain Libraries

Libraries to build and submit Cardano transactions, interact with smart contracts, and query the blockchain.

#### Lucid (Not Maintained)

Lucid is a library to build Cardano transactions and write off-chain code in JavaScript and Typescript. It is designed for simplicity, allowing developers to describe actions at a high level.

#### PyCardano

PyCardano is a lightweight, standalone Cardano client library written in Python. It enables developers to create and sign transactions on the Cardano blockchain without relying on third-party serialization tools. For examples, check out [awesome PyCardano](https://github.com/B3nac/awesome-pycardano).
