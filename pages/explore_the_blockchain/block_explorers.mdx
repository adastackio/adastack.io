# Cardano Block Explorers

Blockchain Explorers give a view of everything happening on the blockchain. Explore transactions, blocks, epochs, staking, metadata, tokens, pools, and more.

#### [CardanoScan](https://cardanoscan.io/)

Cardanoscan is a powerful, fully-featured Cardano explorer built by the team that created [Flac Finance](https://twitter.com/FlacFinance) and [Typhon Wallet](https://twitter.com/TyphonWallet). Support development by delegating to [CSN1](https://cexplorer.io/pool/pool10hex9l4wjgqarvhr94xgyh9frv5lhtajhrj4dah0kl65jtws30khttps://) and [CSN2](https://cexplorer.io/pool/pool1mfgqn8n658vjdcvg3xgtr3qyet64fht0dzsukqeznxw36z8g2k6https://).

#### [AdaStat](https://adastat.net/)

Adastat is another great Cardano explorer. The site has a rich dashboard for viewing [stake pool stats](https://adastat.net/pools). They also display the results of [on-chain votes](https://adastat.net/polls). You can support AdaStat's development by staking to [STAT](https://adastat.net/pools/77b0a93c26ac65be36e9a9f220f9a43cbc57d705fc5d8f1de5fdeea1).

#### [Eutxo.org](https://eutxo.org/)

Eutxo.org is a unique blockchain visualizer tool. You can search for specific blocks in history to see a visual representation of everything that happened within that block. Support the project by making a [donation](https://eutxo.org/contact).

#### [Cardano Explorer](https://explorer.cardano.org/)

Cardano Explorer is an IOHK supported project. It is a straightforward app to search epochs, blocks, addresses, and transactions. It is also [open source](https://github.com/input-output-hk/cardano-explorer-app).
[![Github link](https://img.shields.io/badge/GitHub-100000?style=for-the-badge&logo=github&logoColor=white)](https://github.com/input-output-hk/cardano-explorer-app)

#### [AdaPools](https://adapools.org/)

Adapools is a lightweight version of Cexplorer that still includes the important things you'd expect from a blockchain explorer.
[![Github link](https://img.shields.io/badge/GitHub-100000?style=for-the-badge&logo=github&logoColor=white)](https://github.com/cardanians/adapools.org)

#### [Raw Cardano](https://rawcardano.app/)

Raw Cardano is a useful block explorer for developers because it returns results in a format called json that is used in computer programs.
Support the creators at [QXT](https://twitter.com/quixotepool) and [PIADA](https://twitter.com/Piada_stakePool).

#### [Blockchair](https://blockchair.com/cardano)

Blockchair is a "universal blockchain explorer and search engine" that works automatically with many blockchains including Cardano.

#### [3xpl](https://3xpl.com/cardano)

3xpl is a multi-blockchain explorer that supports Cardano. It works well but lacks some in-depth features.
